# Пример полного отчёта Resume Fire

> Обезличенный Human + ATS-разбор реального backend-резюме. Имя, контакты и идентифицирующие сведения удалены.

## Контекст

- Целевая роль: Senior PHP Backend Developer / Tech Lead.
- Human: 84 из 100.
- ATS: 81 из 100.

## Сильные стороны

- Ясное позиционирование под Senior PHP Backend Developer / Tech Lead.
- Измеримые результаты: до 30 000 jobs/sec, снижение производственных дефектов на 70-80%, обращений в поддержку на 90%.
- Стек PHP, Laravel, RabbitMQ, Redis, MySQL и ClickHouse связан с реальными задачами.
- Наблюдаемость подтверждена использованием Prometheus и Grafana.

## Приоритетные правки

1. Указать размер команды, формат технического лидерства и архитектурную ответственность.
2. В 2-3 строках закрепить целевую роль и масштаб задач.
3. Привести даты и ранний опыт к единому формату.

## Пример изменения

Было: «Разрабатывал backend и улучшал качество продукта».

Стало: «Спроектировал и развивал backend-сервисы обработки задач до 30 000 jobs/sec, обеспечив наблюдаемость через Prometheus и Grafana».

Формулировка использует только факты исходного резюме. Сервис не добавляет неподтверждённые достижения.
